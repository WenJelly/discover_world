# 个人主页动态普通列表设计

## 目标

将个人主页“动态”页签中的时间线展示改为普通动态列表，减少与内容无关的视觉装饰，让动态卡片成为页面主体。

## 界面设计

- 移除时间线左侧竖线、日期节点和动态节点。
- 移除“今天”“昨天”和具体日期等日期分组标题。
- 动态卡片按接口返回的既有顺序直接纵向排列。
- 卡片之间继续使用当前列表间距，不调整 `PostCard` 内部样式。

## 实现边界

- 保留 `PostTimeline` 组件及其对外属性，避免扩大改动范围。
- 将组件内部简化为普通有序列表，每条动态直接渲染一个 `PostCard`。
- 保留列表项的 `data-post-id`，发布新动态后仍可定位并滚动到对应卡片。
- 保留置顶、删除、更新、可见性和评论等现有回调与行为。
- 不修改接口、后端排序、空状态、发布入口或其他个人主页页签。

## 验证

- 运行前端 TypeScript/Vite 构建。
- 运行前端 lint。
- 检查个人主页动态页签的桌面端和移动端布局，确认没有时间线元素、日期分组或新增横向溢出。
- 按用户确认，本次不新增自动化测试。
